# Domain Name Brainstorming Guide

Generate creative domain names and check availability.

## When to Use

- Starting a new project or company
- Launching a product or service
- Creating a personal brand
- Finding alternatives when first choice is taken

## What Makes a Good Domain

| Criterion | Why |
|-----------|-----|
| Short (<15 chars) | Easy to type and remember |
| Memorable | Sticks in people's minds |
| Pronounceable | Can share verbally |
| Descriptive | Hints at what you do |
| No hyphens | Easier to communicate |
| Brandable | Unique enough to stand out |

## TLD Guide

| TLD | Best For |
|-----|----------|
| .com | Universal, trusted, businesses |
| .io | Tech startups, developer tools |
| .dev | Developer-focused products |
| .ai | AI/ML products |
| .app | Mobile or web applications |
| .co | Alternative to .com |
| .xyz | Modern, creative projects |
| .design | Creative/design agencies |

## Pricing Context

| Type | Price |
|------|-------|
| Standard (.com) | $10-15/year |
| Premium TLDs (.io, .ai) | $30-50/year |
| Taken domains | Market price |
| Premium domains | $100s-$1000s |

## Tips for Success

1. **Act fast** - Good domains get taken quickly
2. **Register variations** - Get .com AND .io
3. **Avoid numbers** - Hard to communicate
4. **Check social media** - Ensure @username available
5. **Say it out loud** - Test pronunciation
6. **Check trademarks** - No legal conflicts
7. **Think long-term** - Will it make sense in 5 years?

## Workflow

1. Describe your project and target audience
2. Get 10-15 domain suggestions
3. Review availability and pricing
4. Pick top 3 favorites
5. Register immediately (they go fast!)

## After Picking a Domain

- Check logo design options
- Verify social media handles
- Research trademark availability
- Plan brand identity
